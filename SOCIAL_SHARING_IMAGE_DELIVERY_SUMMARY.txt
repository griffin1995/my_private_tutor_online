================================================================================
COMMON MISTAKES TO AVOID
================================================================================

Design Phase:
✗ Wrong dimensions (not exactly 1200×630px)
✗ File too large (exceeds 300 KB target)
✗ Text extends beyond 60px safe margins
✗ Logo stretched or distorted
✗ American English instead of British English
✗ Missing or illegible checkmarks

Implementation Phase:
✗ Forget to update metadata in layout.tsx
✗ Wrong file path (public/images instead of /images/graphics)
✗ Forget to commit image to git
✗ Don't test with social debuggers
✗ Don't verify meta tags in HTML source
✗ Inconsistent alt text

Testing Phase:
✗ Only test with one platform (test all major platforms)
✗ Don't check file size after deployment
✗ Assume old image cached (use "Scrape Again" in debuggers)
✗ Don't verify Core Web Vitals after deployment
✗ Skip manual WhatsApp/Discord testing
